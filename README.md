# 🌾 Warsaw Valley

> Уютный сайт в стиле **Stardew Valley** × **веб нулевых**, живущий на пиксель-арт карте
> **Варшавы**. Персонажи — это реальные датчики качества воздуха [AirMQ](https://airmq.cc),
> а живые данные (температура, влажность, PM1/2.5/10) управляют погодой и настроением мира.

## ✨ Идея одним абзацем

Заходишь на сайт — тебя встречает загрузочный экран «à la 2003» (бевелы, счётчик
посещений, marquee). Дальше открывается пиксельная карта Варшавы. По ней ходят твои
персонажи: **Fiodar H** и **Dwoch Mieczy**. Кликаешь на персонажа — видишь, какой прямо
сейчас у него воздух: тепло/холодно, влажно/сухо, чисто/пыльно. Если PM высокий — над
районом висит смог; если датчик offline — персонаж «спит дома».

## 🚀 Быстрый старт

```bash
# 1. Установить зависимости (только dev-сервер)
npm install

# 2. Прописать токен AirMQ (секрет!)
cp config/env.example.js config/env.local.js
#   → впиши свой AIRMQ_TOKEN внутрь env.local.js

# 3. Запустить дев-сервер
npm run dev
#   → открой http://localhost:5173
```

**Без Node (быстрое превью через Python):**
```bash
cd src
py -m http.server 8777
#   → открой http://127.0.0.1:8777
```
Это отдаёт статику, ES-модули и `fetch` работают. Прямое открытие `index.html`
через `file://` НЕ годится — браузер заблокирует модули (нужен любой локальный сервер).

## 🔒 Безопасность

Токен AirMQ — это **секрет**. Он лежит только в `config/env.local.js`, который **не
коммитится** (см. `.gitignore`). Для публичного деплоя не встраивай токен в браузерный
код — используй лёгкий прокси (инструкция в [docs/API.md](docs/API.md)).
