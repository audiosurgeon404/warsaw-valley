# Reference art — визуальный канон персонажей

Сюда кладём эталонные картинки, по которым рисуем спрайты и тайлы.
Файлы-референсы НЕ обязаны быть пиксель-перфект — это ориентир по палитре,
композиции и деталям.

## Ожидаемые файлы

- `dwoch_mieczy.png` — двор «Dwoch Mieczy» (закат, дом, пара + 3 питомца).
  → сохрани присланный арт под этим именем.
- `fiodar_h.png` — окно «Fiodar H» (ночь, каменица, тёплое окно, кот).
  → сохрани присланный арт под этим именем.

См. описание канона в `docs/CHARACTERS.md` (раздел «Канон-арт»).
